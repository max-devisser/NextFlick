# NextFlick
UCSB CS48 Student Project - A Java movie database with searching and recommendations

Creators: Artem Jivotovski, Barbara Korycki, Cole Margerum, Max de Visser, Isaac Zinman


Building and Executing:
1. Building works on any OS from the command line
2. Go into the SourceCode directory
3. Compile using javac src/Controller.java
4. Run using java src/Controller


List of known bugs (Things that haven't been implemented)
1. There is no way to remove filters
2. The database contains only five movies
3. Results are cutoff because we don't have scrolling

How to Use:

Click on one of the filters and type in what you what you want to search for.
